# A demo with js+py+c

- js 前端, localhost:5000
- py 后端, localhost:4000
- c 编译生成 dll,由 python ctypes 调用

---

## Usage

```cmd
pip install bottle
```

```powershell
npm install
```

```powershell
npm run dev
```

this will use localhost:5000

```bash
cd backend
```

```powershell
python backend.py
```

this will use localhost:4000
